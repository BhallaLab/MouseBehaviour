# eyeBlinkBehaviour

All codes related to Arduino programming and data saving for the Eye-Blink
Conditioning Tasks

To build and upload to arduino   

    $ make 
    $ make upload 


# Dependencies

- gnuplot-x11
- pip install readchar

# Commands

- __Puff__ : pp
- __CS1__ : 11
- __CS2__ : 22
- __Select__ : ss (mimic SELECT button on LCD/Start trials).
- __Reboot__ : rr
- __Pause__ : ww
- __Unpause__ : cc
